# Modern Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript. Features a clean design, smooth animations, and both light and dark themes.

## 🌟 Features

- **Responsive Design**: Works perfectly on all devices
- **Theme Switcher**: Toggle between light and dark themes
- **Modern UI Effects**: 
  - 3D Parallax backgrounds
  - Particle animations
  - Smooth transitions
  - Interactive cards
- **Project Showcase**: Filterable project grid
- **Skills Section**: Animated skill progress bars
- **Contact Form**: Interactive floating contact form
- **Performance Optimized**: Fast loading and smooth animations

## 🛠️ Technologies

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (ES6+)
- Node.js
- Express.js

## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/ahmertsengol/MyPortfolio.git
   cd MyPortfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the server**
   ```bash
   npm start
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```

## ⚙️ Configuration

1. Edit project data in `frontend/js/main.js`
2. Modify styles in `frontend/public/styles/main.css`
3. Update content in `frontend/index.html`

## 🎨 Customization

### Themes
- Edit theme colors in `:root` and `[data-theme="light"]` in CSS
- Modify theme toggle behavior in `theme` object in JavaScript

### Projects
- Update `projects` array in `main.js` to add/modify projects
- Customize project card layout in CSS

### Skills
- Modify `skills` object in `main.js` to update skills
- Adjust skill card animations in CSS
