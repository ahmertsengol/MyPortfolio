# Modern Portfolio

Ahmet Mert Şengöl'ün kişisel portfolio websitesi. Modern web teknolojileri ile oluşturulmuş, AI/ML ve yazılım geliştirme projelerini sergileyen interaktif platform.

## ✨ Özellikler

- 🎨 Modern ve responsive tasarım
- ⚡ Next.js 15 + React 19 + TypeScript
- 🌟 Partikül animasyonları ve interaktif efektler
- 📱 Mobile-first yaklaşım
- 🔗 GitHub entegrasyonu

## 🚀 Kurulum

```bash
# Projeyi klonlayın
git clone https://github.com/ahmertsengol/MyPortfolio.git
cd MyPortfolio

# Bağımlılıkları yükleyin
npm install

# Geliştirme sunucusunu başlatın
npm run dev
```

Tarayıcınızda [http://localhost:3000](http://localhost:3000) adresini açın.

## 🛠️ Teknolojiler

- **Next.js 15** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animasyonlar
- **OGL** - WebGL efektleri
